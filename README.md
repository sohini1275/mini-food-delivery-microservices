# Mini Food Delivery System

A microservices-based food delivery application built using Node.js and Express.js.

## Services

1. Auth Service (Port 5001)
2. Menu Service (Port 5002)
3. Order Service (Port 5003)

## Features

- JWT Authentication
- Service-to-Service Communication using Axios
- REST APIs
- Microservices Architecture

## Tech Stack

- Node.js
- Express.js
- JWT
- Axios
